# Programming-in-R
Practical R applications for statistical analysis, featuring advanced data manipulation with dplyr and custom data visualizations.

## Featured Project: 
* Titanic Data Preprocessing

## Core Competencies

* **Logic & Syntax:** Arithmetic, logical, and conditional operators and statements.
* **Text Processing:** Advanced string manipulation (`stringr`) and Regular Expressions (Regex).
* **Data Engineering:** Import/Export (CSV, Excel, RDS), Web Scraping (`rvest`), and data cleaning.
* **Functional Programming:** Custom function development and automated computing using the `apply` family.
* **Statistical Analysis:** Leveraging the `tidyverse` ecosystem (primarily `dplyr`) for data exploration.
